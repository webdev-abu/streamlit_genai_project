# 📘 Note Summary and Quiz Generator

## 🧠 Introduction
The **Note Summary and Quiz Generator** is a smart tool designed to help students and professionals quickly understand large volumes of text. It automatically generates concise summaries and creates quizzes to test comprehension.

---

## ✨ Features
- 📄 Automatic text summarization  
- ❓ AI-generated quizzes  
- 📥 Download summaries and quizzes  
- 🌐 User-friendly interface
